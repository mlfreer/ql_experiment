You would be presented with the scanned version of the paper copy of the data book (Peruvian Census, 1876) and the entry field on the right side of the screen. picture below presents the screenshot of the actual screen you will face in this part of the experiment. If you do not see the submit button please scroll the screen down.



The system has also some internal checks to make sure that the data is entered correctly. In a case you have made a typo, the system would alert you and ask to fix a typo. To help you to identify there the typo was, the system would highlight corresponding raw or column. Picture below illustrates the screen you would face in a case system has identified a typo.



Unless most of the typos are fixed the system would not allow you to enter the data and proceed to the next screen. Given that the task is rather complicated we do allow you to do some mistakes. In particular you are allowed for the number of mistakes equal to the number of lines divided by two. For example, if there are 4 lines, then you would be able to make 2 mistakes; if there are 8 lines you would be able to make 4 mistakes. However, to ensure the best possible data collection we would encourage you to take that task responsibly and input data as close to the original as possible. In order to check and fix potential mistakes please check the following control sums (using calculator if necessary):
  • Number in the bottom of every column is the sum of all the numbers above it.
  • In every row, sum of the numbers in columns 10 and 11 must be equal to the number in column 12 (last one).
In addition to the guidelines above please feel free to employ several hints :
  • The ... must be replaced with 0 .
  • Given the font figure looking like 0 can be 6 or 8 .
  • Given the font figure looking like 9 may sometimes be 8 .
  • If number is unreadable (ink-drops), delegate it to the errors allowed and leave it zero


Once you have submitted the data that passes the internal checks the button to proceed will appear as illustrated on picture below. Pressing this button would finalize the submission and allow you to proceed to the next stage of the experiment.
